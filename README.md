## Практическая работа №5

### Структура проекта
- `certs` - все сертификаты для SSL итд
- `config` - содержит конфиги для кластера
- `scripts` - команды для настройки прав и создание топиков
- `src` - код для проверки работоспособности

### Процесс развёртывания
1. Выставить значение для всех брокеров `KAFKA_ALLOW_EVERYONE_IF_NO_ACL_FOUND: "true"`
2. Запустить `docker-compose.yaml`
3. Выполнить ряд операция
   - Все операции указаны внутри `/scripts/commands.sh` с короткими комментами
4. Меняем значение для всех брокеров `KAFKA_ALLOW_EVERYONE_IF_NO_ACL_FOUND: "false"` в `docker-compose.yaml`
5. Перезапускаем контейнеры `docker compose up -d`

### Тестирование
Запустить локально `Main`
